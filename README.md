# 🐍 Snake Game

A classic **Snake Game** built using **Python** and the **Turtle** graphics library. Control the snake, eat food to grow longer, and try to achieve the highest score while avoiding collisions with the walls and your own tail.

---

## 📖 Overview

This project recreates the classic Snake game using Python's built-in Turtle module. It is designed to demonstrate fundamental game development concepts such as object-oriented programming, event handling, collision detection, game loops, and score management.

Whether you're learning Python or exploring game development, this project provides a simple and interactive example of building graphical applications without any external libraries.

---

## ✨ Features

- 🐍 Classic Snake gameplay
- 🎮 Smooth keyboard controls
- 🍎 Random food generation
- 📈 Real-time score tracking
- 🚧 Collision detection with walls
- 💥 Self-collision detection
- 🔄 Automatic game over handling
- 🧩 Modular and object-oriented code
- 🐢 Built entirely with Python's Turtle library

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

Verify installation:

```bash
python --version
```

---

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/snake-game.git
```

2. Navigate to the project directory

```bash
cd snake-game
```

3. Run the game

```bash
python main.py
```

## 🎯 Game Rules

- Eat the food to grow longer.
- Each food increases your score.
- Avoid hitting the walls.
- Avoid colliding with your own body.
- The game ends upon collision.
